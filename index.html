Array Challenges

shift(), unshift(), sort()

  • Mutates the original array

slice()

  • Does not mutate and returns a new array

splice()

  • Mutates the original array
  • Return value and mutated value is different

  • splice() is basically known for deleting array elements

    • Deleted elements are returned in an array
    • Original array does not contain the deleted elements
  •             
              const someName = ['suraj', 'suresh', 'soumen']
              someName.splice(0, someName.indexOf('soumen'))

Object Challenges

every()

  • Does not returns a new array, returns a boolean value
  • Returns true, if the condition is satisfies for each and every element
  • Returns false if the condition is not satified even for one elements

sort()

  • Return val => +ve => Don't Swap
  • Return val => -ve => Swap
